
### 2. Автоматические тесты (2 мин)

1. Нажмите **F12** (Developer Tools)
2. Вкладка **Console**
3. Скопируйте содержимое файла `tests/cms_full_test.js`
4. Вставьте в консоль и нажмите Enter
5. Запустите:

```javascript
runAllCMSTests()
```

**Ожидаемый результат:** Все тесты зеленые ✅

## 🐛 Если что-то не работает

### Проблема: Изменения не видны
**Решение:** Очистите кэш: `Cmd+Shift+R` (Mac) или `Ctrl+Shift+R` (Windows)

### Проблема: Статьи не открываются
**Решение:** Проверьте консоль (F12) на ошибки JavaScript

### Проблема: Не могу войти
**Решение:** Проверьте что сервер запущен и используете правильные данные
