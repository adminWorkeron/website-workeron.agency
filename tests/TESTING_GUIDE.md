## 🚀 Автоматическое тестирование

### Шаг 1: Запуск сервера

```bash
npm start
```

Сервер должен запуститься на `http://localhost:3000`

### Шаг 2: Вход в админ-панель

1. Откройте `http://localhost:3000/admin/`
2. Войдите с учетными данными:
   - **Логин:** admin
   - **Пароль:** workeron2026

### Шаг 3: Запуск автоматических тестов

1. Откройте Developer Tools (F12 или Cmd+Opt+I)
2. Перейдите на вкладку **Console**
3. Откройте файл `tests/cms_full_test.js`
4. Скопируйте весь код и вставьте в консоль
5. Нажмите Enter
6. Запустите тесты командой:

```javascript
runAllCMSTests()
```

### Что проверяют автоматические тесты:

- ✅ **ТЕСТ 1:** Редактирование текстов (API + сохранение)
- ✅ **ТЕСТ 2:** Редактирование изображений
- ✅ **ТЕСТ 3:** Настройки формы обратной связи
- ✅ **ТЕСТ 4:** CRUD операции с блогом (Create, Read, Update, Delete)
- ✅ **ТЕСТ 5:** Синхронизация с фронтендом
- ✅ **ТЕСТ 6:** Проверка безопасности

## 🖱️ Ручное тестирование

### ТЕСТ 1: Редактирование текстов

#### 1.1 Изменение заголовка Hero секции

1. В админ-панели перейдите на вкладку **"Content Editor"**
2. Найдите поле `hero.title`
3. Измените текст на: `"Тестовый заголовок"`
4. Нажмите **"Save All Changes"**
5. Откройте `http://localhost:3000/` в новой вкладке
6. **Проверка:** Заголовок на главной странице должен измениться

#### 1.2 Изменение текста кнопки

1. Найдите поле `hero.cta`
2. Измените на: `"Тестовая кнопка"`
3. Сохраните
4. Обновите главную страницу (Cmd+Shift+R / Ctrl+Shift+R)
5. **Проверка:** Текст кнопки изменился

#### 1.3 Изменение описания услуги

1. Найдите `services.card1.description`
2. Измените текст
3. Сохраните
4. Обновите страницу
5. **Проверка:** Описание первой карточки услуги изменилось

### ТЕСТ 2: Редактирование изображений

#### 2.1 Изменение изображения услуги

1. В Content Editor найдите `services.card1.image`
2. Измените путь на: `img/art2.png`
3. Сохраните
4. Обновите главную страницу
5. **Проверка:** Изображение первой карточки изменилось

#### 2.2 Проверка всех изображений

1. Проверьте что все поля с `.image` редактируются
2. Список ключевых изображений:
   - `services.card1.image`
   - `services.card2.image`
   - `services.card3.image`
   - `services.card4.image`
   - `services.card5.image`

### ТЕСТ 3: Форма обратной связи

#### 3.1 Изменение email получателя

1. Перейдите на вкладку **"Settings"**
2. Найдите поле **"Recipient Email"**
3. Измените на: `test@workeron.ai`
4. Нажмите **"Save Settings"**
5. **Проверка:** В консоли должно появиться "Settings saved successfully"

#### 3.2 Проверка текстов формы (если реализовано)

1. В Settings найдите секцию "Form Texts"
2. Измените тексты полей формы
3. Сохраните
4. Проверьте на главной странице

### ТЕСТ 4: Управление блогом

#### 4.1 Создание новой статьи

1. Перейдите на вкладку **"Blog Manager"**
2. Нажмите **"Create New Article"**
3. Заполните поля:
   - **Title:** `Тестовая статья`
   - **Slug:** `test-article`
   - **Category:** `Testing`
   - **Author:** `Test User`
   - **Excerpt:** `Это тестовая статья`
   - **Image:** `img/art1.png`
4. В редакторе нажмите кнопку **`</>`** (HTML режим)
5. Вставьте HTML:
```html
<h2>Тестовый заголовок</h2>
<p>Это <strong>тестовый</strong> контент с HTML форматированием.</p>
<ul>
  <li>Пункт 1</li>
  <li>Пункт 2</li>
</ul>
```
6. Нажмите **"Save Article"**
7. **Проверка:** Статья появилась в списке

#### 4.2 Проверка статьи на фронтенде

1. Откройте `http://localhost:3000/blog.html`
2. **Проверка:** Тестовая статья отображается в списке
3. Кликните на статью
4. **Проверка:** Модальное окно открывается с правильным контентом
5. **Проверка:** HTML форматирование сохранилось

#### 4.3 Проверка в карусели на главной

1. Откройте `http://localhost:3000/`
2. Прокрутите до секции Blog
3. **Проверка:** Тестовая статья появилась в карусели (если это одна из 3 последних)

#### 4.4 Редактирование статьи

1. Вернитесь в админ-панель
2. Найдите тестовую статью в списке
3. Нажмите **"Edit"**
4. Измените заголовок на: `Обновленная тестовая статья`
5. Сохраните
6. Обновите blog.html
7. **Проверка:** Заголовок изменился

#### 4.5 Удаление статьи

1. В админ-панели найдите тестовую статью
2. Нажмите **"Delete"**
3. Подтвердите удаление
4. **Проверка:** Статья исчезла из списка
5. Обновите blog.html
6. **Проверка:** Статья больше не отображается

## 🔧 Устранение проблем

### Проблема: Изменения не отображаются на сайте

**Решение:**
1. Очистите кэш браузера (Cmd+Shift+R / Ctrl+Shift+R)
2. Проверьте что сервер запущен
3. Проверьте консоль на ошибки
4. Убедитесь что открываете `http://localhost:3000`, а не `file://`

### Проблема: Статьи не открываются

**Решение:**
1. Проверьте консоль на ошибки JavaScript
2. Убедитесь что `articles.js` загружается
3. Проверьте что функции `getAllArticles`, `getArticleBySlug`, `formatDate`, `getRelatedArticles` определены

### Проблема: Не могу войти в админ-панель

**Решение:**
1. Проверьте учетные данные: admin / workeron2026
2. Проверьте что сервер запущен
3. Проверьте консоль сервера на ошибки
